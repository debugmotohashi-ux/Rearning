# Rearning UI v3.0 差し替えセット

## 変更内容
- Instagram風の5タブUIへ変更
- ホーム/学習/テスト/実績/自分 の構成へ変更
- トロフィー画面をグリッド型へ変更
- トロフィー詳細モーダルを追加
- 既存の問題データ、トロフィー取得ロジック、パスワードゲート、localStorage保存は維持
- APP_VERSIONをv3.0へ更新

## GitHubでの使い方
1. index.html を上書き
2. trophies フォルダをリポジトリ直下に置く
3. icon系ファイルも必要なら同名で上書き
4. Commit changes
5. Netlify等の再デプロイ後に確認

## 配置
index.html と trophies フォルダは同じ階層に置いてください。
